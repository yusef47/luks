# Lukas Browser AI Extension

## التثبيت

1. افتح Chrome وانتقل إلى: `chrome://extensions/`
2. فعّل **وضع المطور** (Developer mode) من أعلى اليمين
3. اضغط **تحميل إضافة غير مضغوطة** (Load unpacked)
4. اختر مجلد `extension` من Lukas

## الاستخدام

1. اضغط على أيقونة Lukas في شريط الأدوات
2. اكتب المهمة المطلوبة (مثال: "ابحث عن أفضل فنادق دبي")
3. اضغط **ابدأ**
4. شاهد الذكاء الاصطناعي ينفذ المهمة!

## الإيقاف

اضغط **إيقاف** لإيقاف التنفيذ في أي وقت.

## الملفات

| الملف | الوظيفة |
|-------|---------|
| manifest.json | إعدادات الإضافة |
| popup.html/js | واجهة المستخدم |
| background.js | التنسيق والتحكم |
| content.js | تنفيذ الأوامر |
